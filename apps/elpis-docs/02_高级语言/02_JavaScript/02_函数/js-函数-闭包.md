# 闭包函数



## 脑图
```js
● 闭包
  ○ 是什么
    ■ 闭包就是函数嵌套函数，内部函数可以访问外部函数的作用域。
  ○ 优缺点
    ■ 好处
      ● 好处是 可以延长变量生命周期 和 模块化
      ● 当内部函数保持对闭包变量的引用，GC就不能正常回收
      ● 闭包执行，形成私有作用域，不受全局污染，适合模块化开发，commonjs和esmodule都实用了闭包。
    ■ 缺点
      ● 可能导致内存泄露
      ● 不再使用的变量就赋值为null，切断引用让垃圾回收机制回收。
  ○ 讲应用
    ■ 闭包的应用场景有 防抖 节流 柯里化 vue中的data函数也是用了闭包
```
- 闭包是什么



闭包是 函数嵌套函数，内部函数可以访问外部函数的变量。

```js

function a() {
  const tmp = 1;
  // b函数中访问了外层函数的变量 b函数就是一个闭包函数，第2行代码就是闭包区域
  return function b() {
    console.log(tmp); // 保持tmp变量的引用
    return tmp; // 保持tmp变量的引用
  }
}
// 正常情况 函数执行完毕 局部变量都会销毁
```
