# 工厂函数
## 什么是工厂函数？
工厂函数是一种**创建并返回对象或实例的函数**，它不依赖类的`new`关键字或构造函数，而是通过函数调用来创建对象。

## 基本形式
```javascript
// 简单工厂函数示例
function createUser(name, age) {
  return {
    name,
    age,
    greet() {
      console.log(`Hello, I'm ${this.name}`);
    }
  };
}

// 使用工厂函数
const user1 = createUser('Alice', 25);
const user2 = createUser('Bob', 30);
```

## 解决的问题
### 1. **简化对象创建**
当对象创建逻辑复杂时，工厂函数封装了复杂性：

```javascript
// 复杂对象创建
function createProduct(type, basePrice) {
  const product = { basePrice };
  
  if (type === 'electronics') {
    product.taxRate = 0.2;
    product.warranty = '2 years';
  } else if (type === 'clothing') {
    product.taxRate = 0.1;
    product.sizeChart = true;
  }
  
  product.calculatePrice = function() {
    return this.basePrice * (1 + this.taxRate);
  };
  
  return product;
}
```

### 2. **封装私有状态**
工厂函数可以创建具有私有变量的对象：

```javascript
function createCounter() {
  let count = 0; // 私有变量
  
  return {
    increment() {
      count++;
      return count;
    },
    decrement() {
      count--;
      return count;
    },
    getValue() {
      return count;
    }
  };
}

const counter = createCounter();
counter.increment(); // 外部无法直接访问 count
```

### 3. **实现多态性**
根据参数创建不同类型的对象：

```javascript
function createAnimal(type, name) {
  if (type === 'dog') {
    return {
      name,
      sound: 'Woof!',
      makeSound() {
        console.log(`${this.name} says: ${this.sound}`);
      }
    };
  } else if (type === 'cat') {
    return {
      name,
      sound: 'Meow!',
      makeSound() {
        console.log(`${this.name} says: ${this.sound}`);
      }
    };
  }
}
```

### 4. **避免构造函数的问题**
避免构造函数中的`new`关键字问题：

```javascript
// 构造函数忘记使用 new 会导致问题
function Person(name) {
  this.name = name;
}

const p1 = new Person('Alice'); // 正确
const p2 = Person('Bob'); // 错误：this 指向全局对象

// 工厂函数更安全
function createPerson(name) {
  return { name };
}
const p3 = createPerson('Charlie'); // 总是正确
```

## 工厂函数 vs 类
```javascript
// 类的方式
class User {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  
  greet() {
    console.log(`Hello, I'm ${this.name}`);
  }
}

// 工厂函数的方式
function createUser(name, age) {
  return {
    name,
    age,
    greet() {
      console.log(`Hello, I'm ${this.name}`);
    }
  };
}
```

**工厂函数的优势**：

+ 更灵活的对象创建
+ 更好的封装性
+ 无`this`绑定问题
+ 组合优于继承

## 实际应用场景
### 1. **配置对象**
```javascript
function createConfig(config = {}) {
  const defaults = {
    apiUrl: 'https://api.example.com',
    timeout: 5000,
    retry: 3
  };
  
  return { ...defaults, ...config };
}
```

### 2. **缓存系统**
```javascript
function createCache() {
  const cache = new Map();
  
  return {
    set(key, value) {
      cache.set(key, value);
    },
    get(key) {
      return cache.get(key);
    },
    has(key) {
      return cache.has(key);
    }
  };
}
```

### 3. **数据映射器**
```javascript
function createUserMapper(userData) {
  return {
    id: userData.id,
    fullName: `${userData.firstName} ${userData.lastName}`,
    isAdult: userData.age >= 18,
    formattedBirthdate: new Date(userData.birthdate).toLocaleDateString()
  };
}
```

## 总结
工厂函数是一种**创建型设计模式**，主要解决以下问题：

1. **封装复杂对象创建逻辑**
2. **提供更好的数据封装和私有状态**
3. **灵活的对象创建策略**
4. **避免构造函数的相关问题**
5. **支持组合而非继承的设计哲学**

它在现代JavaScript开发中特别有用，尤其是在函数式编程和需要灵活对象创建的场景中。
