# 函数声明 
- 一个函数只做一件事


- 高阶函数：函数的返回值或参数也是函数
- 柯里化 
- 操作型函数：这个函数会修改外部状态外部某个变量
- 取值型函数：这个函数会返回一个值 或 传递一个回调接收参数 

## 创建函数
```js 
// 方式1 字面量创建
function fn() {
    console.log(this) // 函数调用者
    console.log(arguments) // 伪数组对象
    return 0;
} 
// 方式2 匿名函数创建
const fn2 = function () {} 

// 方式3 箭头函数创建
const fn3 = () => {}

// 方式4 立即执行函数创建 一般用于初始化
const fn4 = (function() { /* 函数体 */})();

// 方式5 回调函数创建
const fn5 = setInterval(() => {}, 1000);

// 方式6 给window绑定click事件
window.onclick = function() { console.log('window123') }

// 方式7 对象
const obj = { fn() {} }

// 方式8  构造函数方式 把类实例化为对象 (应用场景是一些底层库封装才用）
new Function('形参1','形参n','函数体') 
 
```

## 函数调用
```js

/* 写出 5种调用函数的方式 */
fn() 
fn.bind(this)()  // 不需要立即调用就用bind 
fn.apply(this, [])
fn.call(this, 12, 3)
new fn(); // new调用

```
