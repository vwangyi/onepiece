# 外边距塌陷



外边距塌陷是同一个 BFC 中两个块级元素的外边距在垂直方向上发生重叠。 
一般有两种情况，相邻盒子和嵌套盒子会存在外边距塌陷
相邻盒子 A盒子的margin-bottom和 B盒子的margin-top 重叠
嵌套盒子 A盒子的margin-top 和 B盒子的margin-top 重叠
外边距计算规则：如果两个margin的符号相同 那就是绝对值最大的数 如果是一正一负那就是 相加的和
外边距塌陷的解决方案是 给其中一个盒子触发BFC 让两个盒子分开 就可以解决外边距塌陷问题



- 外边距塌陷

- 外边距塌陷是在同一个BFC中(html) 两个块级元素的外边距 在垂直方向上 发生重叠

- 一般有2种情况

- 相邻盒子 A盒子的margin-bottom和 B盒子的margin-top 重叠
- 嵌套盒子 A盒子的margin-top 和 B盒子的margin-top 重叠

- 外边距计算规则

- 两个正数 取最大的数
- 两个负数 取绝对值最大的数
- 一正一负 是相加的和

- 解决方法 

- 给其中一个块级盒子 触发BFC 让两个盒子分开

  - 优缺点
    - 这样设计的好处是 大部分情况下 外边距塌陷更美观
    - 比如 在文字排版中，标题和多个段落之间的外边距是均匀的
