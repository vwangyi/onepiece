+ 调研方案
    - rem 方案
        * 1 动态设置html的font-size大小
            + 1 js设置  
                - 可以用vw 省去这一步
                - vw 就是 视口宽度 / 100
                - htmlDOM.style.fontSize = 视口宽度（ htmlDOM.clientWidth / 10。 也就是 vw单位）
            + 3 lib-flexible库
            + 2 媒体查询
        * 2 根据设计稿 px 计算 对应的rem 
            + 1 手动计算
                - 设计稿 盒子的 xx px / html的fons-size的大小 = xxx rem // font-size的大小是html的width / 10
            + 使用 less或scss 函数 来判断 只需要传入 参数即可
            + 3 vscode 插件（px to rem & rpx &vw） 计算
                - 插件的配置里面 设置 font-size 大小
                - 写px单位 会自动提示 对应的rem值
            + 4 webpack 插件 （postcss-pxtorem） 自动把px转rem
                - webpack插件 设置 font-size 大小
                - webpack 会自动把px转rem
    - vw 方案
        * 1 手动计算
            + 100px = 100 / 设计稿width（375 或 750 ） / 100 = 2.667 vw  
            + width height. 都是 用vw单位
        * 2 less scss 函数
        * 3 vscode 插件
        * 4 webpack插件  postcss-px-to-viewport-8-plugin
        * 好处 
            + 不需要计算font-size
        * 高度
            + height: vw
                - 即使是高度 也是采用 根据宽度换算的 vw
