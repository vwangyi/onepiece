

# 标准流布局



```js
/* 布局方式 /

display: block;
display: inline;
display: inline-block;  

## 标准流-块盒布局
display: block;
块盒特性： 
1. 块盒独占一行,从上到下依次排列
2. 块盒内部可以放 所有类型的盒子
## 标准流-行盒布局
display: inline;
特性：行级元素不会独占一行，从左到右依次排列。行满另起一行。

1. 行盒内部只能放行盒和文字

## 标准流-行级块盒布局
display: inline-block;
特性：多个行级块元素 之间有换行或多个空格 都会产生一个空格
1. 是
```

## 行级、块级、行级块

- 块级元素 独占一行 display 默认为 block, width默认 100%, 常用标签 div, 修改display可以转换类型。 
- 行级元素不会独占一行 行满自动换行，display为inline， 不可以设置width height 只能被内容撑开 不能设置垂直方向的marginpadign，边框 背景正常
    （多个span之间有 换行或空格 会导致页面上也有一个空格 解决方案 给父盒子设置fontsize为0 然后在子盒子上恢复fontsize大小）
    实际场景都会 压缩html 过滤掉换行空格 所以这种情况不存在
- 行级块元素 就是 不会独占一行的块级元素 
    三种标签：
    块级标签 display 默认为 block, 代表 标签 div 
    行级标签 display默认为 inline ， 常用标签 span 和 a 标签
    行级块标签 display 默认为 inline-block 常用标签 img input areatext 等
- 块级元素 可以放 块级或行级
- 行级元素 只能放行级或文本
- 

```js
.box {
  display: inline;  /* 行级元素 */
  /* 
    行级元素特点： 

    1、前后无换行符 （相邻的行内元素在一行上，一行可以有多个行内元素，但之间有空隙（可以设置负margin值解决，设置的大小各浏览器不一））；

    2、设置的宽和高是不起作用，元素大小根据元素内文本决定； 竖直方向的margin无效，水平有效；竖直和水平方向的padding都有效（竖直方向的padding有效，但不会撑开周围的盒子）

    3、行内元素里只能放文本或者行内元素；（a标签里可以放块级元素） 

          1 行内元素: span a i  
      2 行内元素不独占一行 行满自动换行  
      3 直接给宽高无效  默认内容宽高   
      4 只有水平方向的边距有效
    5 行内元素内部 只能放 文本或其他行内元素

    (两个行内标签之间有换行或多个空格  会解析为一个空格显示在网页上) 


  */


  display: block;  /* 块级元素 */
  /* 
    块级元素特点： 
    1、前后都有换行符 （自己独占一行）；

    2、高度、宽度、内外边距各方向可以自由控制；

    3、默认宽度是父容器的100%;

    4、里面可以放行内元素、行内块元素或块级元素;(文字类标签里面不能放块级元素) 
          块级元素独占一行 可以给宽高内外边距 默认宽度为父亲的 width：100%  默认高度0 有内容则是内容高度 
      
      内部可以放块级和行内

      文字类的块级 内部不能放块级 比如 p h1-h6 
  */
  display: inline-block;  /* 行级块元素 */
  /* 
    行级块元素特点： 
    1、前后无换行符 （相邻的行内元素在一行上，一行可以有多个行内元素，但之间有空隙（可以设置负margin值解决，设置的大小各浏览器不一））；

    2、设置的宽和高是起作用，元素大小根据元素内文本决定； 竖直方向的margin无效，水平有效；竖直和水平方向的padding都有效（竖直方向的padding有效，但不会撑开周围的盒子）

    3、行内元素里只能放文本或者行内元素；（a标签里可以放块级元素） 

          行内块的特征 宽度自适应  行内块元素可以当字看

      行内块 （不换行的块级元素）
      行内块 可以宽高  不独占一行 

      只有水平方向的边距有效

      行内元素只能放 文本或其他行内元素

      (两个行内标签之间有换行或多个空格  会解析为一个空格显示在网页上) 
      这个空格需要处理 
      1 浮动 
      2 父亲font-size: 0 孩子重新给font-size
      3 百度

      一般都是 给行内元素转换为行内块元素  或者 块级元素 转换为 行内块

  */
}
```
