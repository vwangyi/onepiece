## 包执行相关配置

- *main*：代表包的入口文件

- browser：该选项表示如果是在浏览器环境下，可以替换一些特定模块或者文件。

  - 指定浏览器的入口文件

  ```json
  {
    "main": "index.js",
    "browser": "browser.js"
  }
  ```

  上面的配置表面在 node 环境下，index.js 为入口文件，如果是浏览器环境，使用 browser.js 作为入口文件

  - 替换特定的模块：浏览器环境时 用browser-version文件替换 node-version

  ```json
  {
    "browser": {
      "./node-version.js": "./browser-version.js"
    }
  }
  ```

  - 排除某些模块：浏览器环境打包时，fs模块排除在外不用打包 因为浏览器环境不能用fs

  ```json
  {
    "browser": {
      "fs": false
    }
  }
  ```
