# 链表

- 链表是什么
- 链表是一种线性数据结构，元素在内存中不连续存储，每个元素包含数据和指向下一个元素的指针。

- 单向链表：每个节点中 只保存 自身数据和下一个数据的引用指向。
- 双向链表：每个节点中 既保存 下一个引用地址 也保存上一个引用地址  头节点的上一个引用是null 尾节点的下一个也是null 
- 循环链表：将尾节点的下一个地址 保存为 头节点 形成一个环 

记住：链表的所有操作本质上都是指针的重新指向。掌握这一点，链表就不再难了！
