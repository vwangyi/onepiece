## 数据类型
- 数据为什么要分类型，因为内存是有限的，不同类型占用内存大小不一样。给数据分门别类。
- 数据类型分为2类：原子类型：不可再分的基本类型。结构类型：可以再分的组合类型。


## 算法
- 算法是 描述解决特定问题的方法

- 算法的5个特性：输入 输出 有穷性 确定性 可行性 
- 输入可以为0 输出至少1个
- 有穷性就是 有边界 不能死循环
- 确定性，每一步都是确定的 不能出现二义性，不能模糊不清。
- 可行性

## 好的算法
- 正确性：结果要正确
- 可读性：代码可读性好
- 健壮性：当输入数据不合法，也能正确处理。
- 时间效率高，占用存储底
