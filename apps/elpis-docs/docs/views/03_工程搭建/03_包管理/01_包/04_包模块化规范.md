# 包模块化规范  
- type是node提供的字段 而不是 npm。所以需要去node官网查看type的可选值。而不是去npm官网查看。
```json
{
  "type": "commonjs", // 表示整个包默认采用commonjs 某些模块想用esm 那就定义.mjs
  "type": "module", //  表示整个包默认采用esmodule 某些模块想用commonjs 那就定义.cjs
}
``` 

- 浏览器环境和 node环境（nodejs v12+） 都支持 esm 规范

- .mjs 也表示 当前js是es module模块规范
- .cjs 也表示 当前js是commonjs模块规范
