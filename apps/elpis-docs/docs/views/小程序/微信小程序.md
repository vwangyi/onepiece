npm包是 是否支持小程序 取决于 编写 npm包的人有没有兼容微信小程序

wx 环境里面 没有浏览器的 window 对象 那么 vant-weapp 这个 npm 包就是支持微信小程序的，但是是以 npm 包的方式 提供给 微信小程序项目使用的 

并不是所有 npm包都可以放到 微信小程序里面使用 

那也就是说 微信小程序项目 需要使用 vant weapp ，那就必须 npm init 作为一个 npm 项目

因为 vant weapp 是以 npm 包发布的 微信小程序也有一个 package.json  

微信小程序也是支持 npm 的 所以 vant weapp 才可以用 npm 包提供
