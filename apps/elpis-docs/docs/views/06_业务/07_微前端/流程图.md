#

## 1 拿地

拿地：签订土地出让合同 & 取得不动产权证
审批部门： 自然资源局（原国土局）公共资源交易中心
核心材料：
• 土地竞买保证金缴纳凭证
• 竞得确认书
• 土地出让合同
• 土地价款缴清证明

常见痛点：
• 土地权属纠纷或历史遗留问题
• 出让合同条款苛刻（如开工时间紧）
• 竞买保证金占用大量资金

## 2 拿 建设用地规划许可证

审批部门：自然资源局（规划部门）
核心材料：
• 土地出让合同
• 项目立项/备案文件（发改委）
• 1:500或1:1000现状地形图
• 规划条件通知书

常见痛点：
• 规划条件与土地出让合同不一致
• 地形图更新慢，需多次补测
• 各部门对用地边界理解不同，反复退件

## 3 拿 建设工程规划许可证

审批部门： 自然资源局（规划部门）人防、消防、园林等部门（并联审批）
核心材料：
• 建筑设计方案（总平图、单体平立剖）
• 日照分析报告
• 人防、消防、绿化等专项审查意见
• 放线报告

常见痛点：
• 方案反复修改（容积率、退线、车位配比等）
• 各部门意见相互矛盾（如消防通道与人防口部冲突）
• 图纸版本混乱，导致审批退回

## 4 拿 建筑工程施工许可证

审批部门： 住建局（行政审批科）
核心材料：
• 建设工程规划许可证
• 施工图设计文件审查合格书
• 施工合同、监理合同
• 建设资金落实证明
常见痛点：
• 施工图审查耗时长（尤其消防、结构超限）
• 施工单位资质或业绩不满足要求
• 资金证明需要银行实时验资，手续繁琐

## 5 拿 商品房预售许可证

审批部门： 住建局（房地产市场监管处/科）
核心材料：
• 施工进度证明（如：投入开发建设的资金达到25%以上，或主体结构达到一定层数）
• 预售方案（包括价格、开盘时间等）
• 前期物业合同
• 商品房预售资金监管协议

常见痛点：
• 施工进度达不到法定要求，开发商赶工被查
• 预售方案中的价格被物价部门干预
• 资金监管账户开立慢，影响拿证速度

一、使用者及价值
使用者 具体角色 创造的价值

房地产开发企业（报建员、项目负责人） 申请方 • 节省时间成本：从线下跑腿、重复交材料变为线上一次提交，审批周期从数月缩短至数周
• 降低合规风险：系统自动校验材料完整性、进度透明，减少因流程不清导致的延误或处罚
• 可追溯：每次驳回、补正都有记录，便于内部复盘和改进
审批政府部门（住建局、自然资源局等科室人员、领导） 审批方 • 提升效率：并行审批、自动流转替代纸质文件传递，单个审批事项从几天压缩到几小时
• 规范权力：流程固定、超时催办、全程留痕，减少人情干扰和廉政风险
• 数据决策：汇集所有项目审批数据，可分析区域开发热度、审批瓶颈，优化政策
系统运维/监管方（大数据局、政务办） 管理方 • 统一监管：实时监控每个审批环节耗时，发现堵点并督办
• 跨部门协同：打通自然资源、住建、消防等部门数据，实现“一表通办”
• 量化考核：自动生成各科室审批时效排名，推动服务改进
社会公众/购房者（间接使用者） 受益方 • 信息透明：可公开查询预售许可证真伪、项目审批进度，减少购房风险
• 监督作用：通过公示环节参与社会监督（如违规预售举报）

核心价值总结（一句话）：将传统“人跑、纸跑”的串联审批，变为“数据跑、系统跑”的并联审批，让企业少等一天、政府少盖一章、群众少担一分风险。
