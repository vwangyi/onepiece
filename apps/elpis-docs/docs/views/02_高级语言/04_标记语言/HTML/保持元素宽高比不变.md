

## 是什么
当元素宽度变化。高度相应变化 但保持比例不变



## 应用场景
渲染 video 视频。往往要求保持元素宽高比不变





## 方案一
+ 缺点是 兼容性不好

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
      .item {
        background-color: red;
        width: 50%;
        margin: 0 auto;
        /* height: 300px; */ 
        /* 使用aspect-ratio属性 就不能使用height了 因为 通过width和aspect-ratio就可以计算出高度 */
        aspect-ratio: 16 / 9; /* 宽度为16 高度为9 */
      }
    </style>
  </head>
  <body>
    <div class="item"></div>
  </body>
</html>
```

## 方案二
+ 好处是 兼容性好

```javascript
<!DOCTYPE html>
<html lang="en">
<head> 
    <style>
        .item {
            background-color: red;
            width: 50%;
            margin: 0 auto;
            /* height: 300px;   */
        }

        .inner {
            position: relative;
            width: 100%;
            /* 这里使用padding而不是height 有细节。  重要： padding的4个方向的属性的百分比取值 都是相对于 包含块宽度的 */
            /*padding-top：75% 的意思是 设置高度为包含块的width的75% */
            padding-top: 75%;  /* 75% 也就是 宽高比为4比3  */

            /* height: 包含块的width的75%; */

        }

        /* inner里面 使用的是 padding-top 那么height就是0 为了可以放其他元素 所以加了一个 绝对定位的盒子 这里面就可以放其他元素了 */
        .container {
            position: absolute;
            inset: 0; /* inset:0; 是尺寸填满 */
            background-color: rgb(70, 144, 98);
        }
    </style>
</head>
<body>
    <div class="item">
        <div class="inner">
            <div class="container">

            </div>
        </div>
    </div>
</body>
</html>
```
