在 JavaScript 中，**表达式** 是指一段能够被计算并 **产生一个值** 的合法代码单元。

简单来说：**凡是可以被求值，并且能返回一个结果的地方，就是表达式**。

## 1. 表达式的特征

-   **一定会返回一个值**（即使这个值是 `undefined`）
-   **可以写在任何需要值的地方**（如函数参数、赋值右侧、模板字符串等）

## 2. 常见表达式类型

| 类型 | 示例 | 结果值 |
| :--- | :--- | :--- |
| **字面量** | `42`、`"hello"`、`true`、`null`、`[1,2,3]`、`{a:1}` | 本身的值 |
| **变量** | `x`、`count`、`user.name` | 变量存储的值 |
| **算术** | `10 + 5`、`a * b`、`3 > 2` | 运算后的值 |
| **赋值** | `x = 5` | `5`（赋值表达式返回右侧值） |
| **函数调用** | `Math.max(1,2)`、`console.log('hi')` | 函数的返回值 |
| **三元运算符** | `age >= 18 ? '成年' : '未成年'` | 满足条件时的那个值 |
| **逻辑运算** | `true && false`、`a \|\| b` | 最终决定的那个值 |
| **成员访问** | `obj.name`、`arr[0]` | 取到的属性值 |

## 3. 表达式 vs 语句（重要区别）

| 对比 | **表达式** | **语句** |
| :--- | :--- | :--- |
| 核心目的 | 产生一个值 | 执行一个动作（如循环、判断） |
| 能否赋值给变量 | ✅ 可以：`let a = 10 + 5` | ❌ 不可以：`let a = if(x>0){...}` 会报错 |
| 示例 | `2 + 2`、`x > y`、`func()` | `if...else`、`for`、`while`、`return` |

> **关键点**：表达式可以嵌套在更大的表达式中，但语句不能嵌套在表达式中。

## 4. 在 Vue 模板中的特殊说明

Vue 模板中的插值表达式 `{{ }}` **只支持单个 JS 表达式**，不支持语句。

```vue
<!-- ✅ 正确：表达式 -->
{{ message }}
{{ count + 1 }}
{{ isActive ? '激活' : '未激活' }}
{{ user.name.split('').reverse().join('') }}

<!-- ❌ 错误：语句 -->
{{ if (isActive) { return '激活' } }}  <!-- if 是语句 -->
{{ let a = 10 }}                       <!-- let 是语句 -->
{{ for (let i of list) {} }}          <!-- for 是语句 -->
```

## 5. 几个容易混淆的例子

```javascript
// 1. 花括号：表达式还是语句？
{ a: 1 }           // 两种情况：
                   // - 在表达式位置：对象字面量，值为 {a:1}
                   // - 在语句位置：代码块，里面的 a:1 是标签

// 2. 逗号运算符（表达式）
let result = (1 + 2, 3 + 4)   // result = 7（返回最后一个表达式的值）

// 3. 函数声明（语句）vs 箭头函数（表达式）
function foo() {}    // 语句，不能直接用在需要值的地方
const bar = () => {} // 右侧是箭头函数表达式，可以
```

## 总结

**表达式 = 能算出值的代码**。记住一个简单判断方法：**能否作为函数参数传入？** 能，大概率就是表达式。比如 `console.log(2 + 2)` 可以，但 `console.log(if (true) {})` 不行。
