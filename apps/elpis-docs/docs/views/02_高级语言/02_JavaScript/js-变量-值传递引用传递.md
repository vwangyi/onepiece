## 值传递

- 值传递是什么
- 值传递就是在传递参数的时候，这个变量的值是什么就传递什么，值是原始值就把原始值 的副本 给变量 B，值是一个引用地址，就把引用地址给 变量 B，变量 A 和变量 B 是完全独立的，当给变量 B 重新赋值，就是切断了引用, 不影响变量 A。
- 变量 B.xxx = {} 这不叫改变 B ，B 的内存地址没变。





## 引用传递
```cpp
#include <iostream>
using namespace std;

int a = 10;
int fn(int &b) 
{
    b = 100; 
    // 变量a和变量b是同一个内存 修改B也就是重新赋值 A也会改变
    cout << "调用后(值传递): a = " << a << endl;  // 100
    cout << "调用后(值传递): b = " << b << endl;  // 100
    return 0;
}

int main()
{ 
    fn(a);
    return 0;
}
// https://www.onlinegdb.com/ 在线执行cpp 
```



## 区别
值传递和引用传递的区别是 变量 A 传递给变量 B，给变量 B 重新赋值，是否影响变量 A。影响就是引用传递，不影响就是值传递。
