## 问：说一下**<font style="color:rgb(38, 38, 38);">值传递和引用传递</font>**


```javascript
值传递：把a变量里面的东西 复制粘贴到（浅拷贝） b变量 
引用传递：把a变量的引用 直接给 b变量

js中是采用值传递 
如果 a变量中是 基础数据类型 直接把值复制粘贴到 b变量
如果 a变量中是 引用数据类型 直接把值复制粘贴到 b变量 （只不过这里的值是16进制的引用地址）
  a变量和b变量 是存的是同一个引用地址 


引用传递 
两个变量 直接共用同一个地址 

值传递
把a变量里面的东西 复制粘贴到（浅拷贝） b变量 

不管a变量中 存的是 基础数据类型值 还是引用地址 都是值
都是把值 复制粘贴到 另一个变量

虽然a变量存的值是一个引用地址 按"值传递"给 b变量 

b里面的东西改了 a也会受影响 这种现象并不能说明是引用传递 
当 直接给b变量赋值一个新对象 （切断了引用） 这样并不会影响a变量

操作数据 你要明白 你改的是谁 
如果a变量是引用类型 
  直接对a赋值 a = {} 这是修改了a 
  a.xx = {} 这并没有修改a变量 直接修改了a的xx变量
```
