## 技能专长 
1. 熟练掌握 HTML5、CSS3、Less、Sass等前端技术，熟悉盒子模型、外边距塌陷、flex布局等等前端概念。
2. 熟练掌握 JavaScript、ES6+，熟悉闭包、原型、浏览器渲染原理、事件循环、Promise、垃圾回收。
3. 熟练掌握 Vue2、Vue3，熟悉数据响应式和组件化，并研究过 diff 源码，能站在源码角度解决项目中存在的问题。
4. 熟练掌握 Webpack、Vite构建工具，熟悉Webpack构建过程、以及Webpack性能优化，能够进行项目从0到1的搭建。
5. 熟练掌握基于 Echarts 实现数据可视化开发，实现各类图表开发。
6. 熟悉 NodeJS 及其框架 Koa 搭配 Mysql，Knex 等。
7. 了解 Docker、Kubernetes 等容器化技术及 DevOps 流程，可实现CICD持续集成持续部署。
8. 具有业务sense和主动性，深刻明白业务价值才是技术价值的根本。
9. 常看书籍红宝书《JavaScript高级程序设计(第4版)》、阮一峰的《ES6标准入门》等等。
10. AI提效工具 vscode+copilot 和 cursor等
