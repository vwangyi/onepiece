 - 尚尚健康 - 圣乐医疗慢病系统 （ 2021.11 - 2022.04 ）
**项目描述**：该项目是专注慢病医疗管理的PC端SPA单页面项目，对患者饮食、运动、作息进行合理规划和控制。患者需要合理的管理模式来
帮助他们完成治疗方案、加强自我管理。
- **技术栈**：Vue、Vuex、Element-UI、Vue-Router。
- **团队规模**：研发14+人 前端6人。
- **负责内容**：
1. 完成 患者信息、慢病监测、病床管理、用药提醒、随访跟踪、健康指导 等模块。
2. 针对接口、网络类型复杂的情况，将axios二次封装增加统一携带token，统一错误处理，降低人工成本，统一请求标准化。
3. 采用路由懒加载 按需加载，提高页面的响应速度，首屏加载速度，提升了用户体验。
4. 针对前后端分离跨域问题，开发环境采用本地Node代理服务器解决，生产环境采用Nginx服务器代理解决，区分不同环境的实施方案。
5. 针对图片优化场景，封装全局自定义指令实现图片懒加载，页面图片相关性能指标提升40%，降低90%用户投诉。
6. 基于平台多角色的复杂权限场景，采用RBAC权限模型设计，结合动态路由与静态路由混合加载方案，实现精细化左侧菜单权限控制。
