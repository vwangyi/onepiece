- 尚尚健康 - 尚尚健康管理系统（ 2022.03 - 2023.08 ）
**项目描述**：该项目是为用户管理全方位健康管理服务的PC端SPA项目，通过科学饮食规划、作息监测及运动指导等功能，高效管理4000+会
员健康数据，日均生成个性化健康方案200+份，助力用户改善亚健康状态。
- **技术栈**：Vue、Vuex、Element-UI、Vue-Router、Echarts。
- **团队规模**：研发14+人 前端6人。
- **负责内容**：
1. 负责 商品 订单 医生 药材 用户下单 医师入驻开处方 等等核心模块功能 。
2. 根据项目中通用选择场景，二次封装可选择的表格组件，结合插槽实现自定组件的结构和内容，支持单选多选反显，提升了开发效率。
3. 针对项目需求业务复杂，引入model层模块化管理数据，提高了项目数据可维护性。
4. 针对复杂的页面DOM结构，同时保证路由一致方便跳转，把条件渲染改为采用component动态组件实现，解决组件代码行数过长，提升
代码可维护性和代码可读性。
5. 针对各个成员没有统一规范，主动开展codeReview会议与业务复盘会，统一团队标准化，总结项目得失，利于大家全局理解业务，统一
规范工作效率提升70%。
6. 针对上传长视频的场景，采用大文件切片上传，增量计算hash 实现断点续传，网络请求并发控制，提高了上传速度，优化了上传体验。
