## 流程控制语句 
- 顺序 分支 循环 
- 顺序：代码‌从上到下、从左到右‌的顺序，一行一行依次执行.
- 分支： if else 和 switch
- 循环： for while 和 do while 

 
循环：侧重于对代码块的重复执行 

遍历”（traversal）则适用于对非线性结构的数据进行特定顺序的访问。例如，在树、二叉树或图等数据结构中，我们可以根据需要选择不同的遍历方式，如先根遍历、中根遍历或后根遍历等。

“迭代”（iterate）特指对线性结构中的数据按照先后顺序进行访问。线性结构如数组，其元素只能从头到尾或从尾到头依次访问，这种访问方式是天然决定的。

“递归”（recursion）是指函数对自身的不断调用。


## 常量
- 常量就是不能修改的变量。 用const声明，引用类型的变量只要引用地址不变就不算修改。
## 标识符
标识符是用来命名变量的符号。由字母 数字 下划线(_)组成， 不能数字开头 不能是关键字保留字，区分大小写。
