
## const/let/var区别 
1. const和let的唯一区别是 const必须赋值且不能修改。
2. let 和 var 的区别是 
3. let是块级作用域  var是函数作用域
4. let不能在声明前访问 会报错暂时性死区  var可以在声明前访问 var有变量提升 
5. 一般是 无脑const 修改用let 不用var 
 
## 变量提升
- 变量提升是 变量声明会提升到顶部，赋值不会提升。
- 函数和变量同名，函数会先提升 变量会覆盖函数。  

## 表达式
- 表达式是 能够计算并产生一个值的合法代码单元，能够作为函数参数传入，基本就是js表达式。


## 数据类型  
数据类型一共是 7个基础类型 1个引用类型。分别是 number string boolean null undefined symbol bigint 和 object。
- 数据有类型，但js变量没有类型，ts的作用就是让js变量有类型。
```js
typeof 1; // 'number'
typeof ''; // 'string' 
typeof true; // 'boolean'
typeof 1n; // 'bigint'
typeof Symbol(); // 'symbol' 
typeof undefined; // 'undefined' 
typeof null; // 'object'  没返回null 这是一个设计缺陷 
typeof function() {}; // 'function'  函数是一个特殊对象
typeof []; // 'object' 
```

## typeof null返回什么
- typeof null 返回 字符串的object 这是一个设计缺陷。
