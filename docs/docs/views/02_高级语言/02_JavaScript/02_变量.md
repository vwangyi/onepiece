## 数据类型
- 数据类型是7个基础类型 1个引用类型：string number boolean null undefined symbol bigint 和 object
- 数据类型有哪些：数据类型是7个基础类型1个引用类型。分别是 number string boolean null undefined symbol bigint 和 object。
- 数据有类型，但js变量没有类型，ts的作用就是让js变量有类型。
```js
typeof 1; // 'number'
typeof ''; // 'string' 
typeof true; // 'boolean'
typeof 1n; // 'bigint'
typeof Symbol(); // 'symbol' 
typeof undefined; // 'undefined' 
typeof null; // 'object'  没返回null 这是一个设计缺陷 
typeof function() {}; // 'function'  函数是一个特殊对象
typeof []; // 'object' 
```

## 变量提升 
- 变量提升：变量提升是声明会提升 赋值不会提升。
- 函数和变量同名，函数会先提升，变量会覆盖函数。
```js
var a = 1; // var变量a
function a() {  // function变量a 
  console.log(a);
  let a = 2;
}
console.log(a); // 输出 1 
a(); // 报错 a不是一个函数

// 函数声明提升到顶部，a 最初是函数
// var a 覆盖了 function a 
// 所以 a为1 a不是一个函数

```
