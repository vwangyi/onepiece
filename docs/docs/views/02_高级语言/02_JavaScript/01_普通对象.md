# Object
- js对象是一个无序的键值对集合，键只能是string或symbol，值可以是任意类型。



- 创建对象

- 对象新增
- 对象删除
- 对象修改
- 对象查询


## 创建对象
```js
const obj = {}; // 底层也是 new Object()
const obj = new Object();
const obj = Object.create({a:1}); // 指定原型对象 obj.__proto__ >>> {a:1}    Object.create 用来实现继承
// Object.create方法 内部做了  实例对象的__proto__ 指向 其构造函数的prototype 



```


## 对象新增
```js 
obj.a = 1; // 新增一个属性a {a: 1}
obj['a'] = 1; // 属性可以是一个变量
```


## 对象删除
```js 
delete obj['age'] // 删除age属性
delete obj // 删除obj对象
```


## 对象修改
```js 
obj['age'] = 200
```

## 对象查询
```js 
obj['age']
obj.age // 链式操作符
obj?.age; // 可选链式操作符
const { age } = obj // 解构赋值 
const { name = '', age = 0 } = { name: 'zs', age: 18 }
const { name: uname } = { name: 'zz' } // 起别名 原名失效 
obj.__proto__  // 原型链开始   Object.prototype.__proto__ 原型链结束

const { a: { b: { c } } } = {a: {b:{c:1}}} // 连续解构


const { children, ...rest } = {a: 1, b:2, children: [] };
console.log(children) // [] 
console.log(rest) // {a: 1, b:2}
```

 

## 对象遍历
```js
for( let key in {a:1}) {
  console.log(key)
} 
```

## 对象合并
```js
Object.assign({}, { a: 1 }, { b: 2 }); // es5
{ ...{ a: 1 }, ...{ b: 2 }} // es6    ...对象 并不是 调用迭代器  这里不做深究
``` 

## 对象冻结
```js
// 对数据 进行冻结操作
const obj = { name: 'zs' }
Object.freeze(obj) // 冻结传入的对象

// 判断对象是否被冻结 返回布尔
const obj = { name: 'zs' }
Object.isFrozen(obj) // false


```
## 对象密封
```js

// 对数据 进行密封操作
const obj = { name: 'zs' }
Object.seal(obj) // 密封传入的对象

// 判断对象是否被密封 返回布尔
const obj = { name: 'zs' }
Object.isSealed(obj) // false
  



```

## 对象判断
```js
Object.prototype.toString === toString // true 因为 window.toString 最终也是通过原型链 继承于Object 
const isPlainObject = obj => Object.prototype.toString.call(obj) === '[object Object]'
const isPlainObject = obj => toString.call(obj) === '[object Object]'
console.log(isPlainObject({}))

// 判断对象某个属性是否存在 

// has 存在
// own 自己
// property 属性


'toString' in { key: null }; // true 检测自身和原型链  
'key' in { key: null }; // true 检测自身和原型链

```

## 对象原型
- 获取对象的原型
```js
({a:1}).__proto__ = {a:2} // 不推荐
({a:1}).__proto__ // es5  不推荐使用
Object.getPrototypeOf({a:1}); // es6 推荐使用 获取原型对象
Object.setPrototypeOf({a:2}); // es6 推荐使用 设置原型对象
```

## 对象属性描述符 Object.defineProperty
```js
// defineProperties 和 defineProperty 区别就是
// defineProperties 可以定义多个 defineProperty用于定义一个（通过循环也可以多个）

// 对数据 进行定义属性操作
const obj = {} 
Object.defineProperty(obj, 'a', {
  value: 1, // 给 属性 a 设置值
  writable: true, // 属性 a 是否可重写
  enumerable: true, // 属性 a 是否可遍历
  configurable: true // 描述符本身 是否可修改
})


const desc = Object.getOwnPropertyDescriptor(obj, 'a') // 获取 属性 a 的描述符
console.log(desc) // { value: 1, writable: true, enumerable: true, configurable: true }
 


// 对数据 进行属性的读写操作
let obj = {}
let internal = undefined
Object.defineProperty(obj, 'name', {
  get: function () {
    return internal
  },
  set(val) {
    internal = val
  }
})
obj.name = 'zs' // 调用set方法
console.log(obj.name) // 调用get方法
```
