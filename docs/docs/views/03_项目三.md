
- 神州数码 - 智能网点运营驾驶舱（ 2023.09 - 2024.05 ）
- **项目描述**：该项目是为各级管理层提供实时监控、核心业务指标的PC端驾驶舱项目SPA项目，日均使用人数200+（含CEO及区域总监）。
- **技术栈**：Vue3、Pinia、Echarts、Vue-Echarts、Ant-Design-Vue、Webpack5。
- **团队规模**：研发14人，前端5人。
- **负责内容**：
1. 项目负责人，基于webpack从 0 到 1 的搭建，经历整个生命周期，调研方案可行性，难点攻克。
2. 针对业务的实时监控场景，采用WebSocket方案实现数据动态推送，相比传统轮询机制，将数据延迟降低70%，确保客流数据的实时性
和准确性。
3. 配置并管理 Nginx 服务器，实现高效的静态资源托管，确保项目在生产环境中的稳定性与性能。
4. 基于可视化项目中的 ECharts 封装可复用的可视化组件，通过参数化配置和动态数据驱动，减少重复代码 60%+，显著提升前端开发效
率，并确保多模块的图表风格一致。
5. 采用Pinia进行集中式状态管理，通过模块化设计高效维护复杂业务数据，显著提升项目可维护性与数据流可控性。
6. 针对总行机构海量下属分支机构数据场景，采用前端虚拟列表滚动方案，实现上万条数据秒级加载，页面响应速度提升80%，大幅提升分
行渲染效率。
