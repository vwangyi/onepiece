## 最短路径

```js
// 从A到D的最短路径 = min(从A到B的最短 + B到D, 从A到C的最短 + C到D)
function shortestPathDP() {
    /*
    假设路线图：
        A --3--> B
        A --1--> C
        B --1--> D
        C --4--> D
    
    从A到D的最短路径？
    
    动态规划思想：
    1. 从A到B的最短：3（直接）
    2. 从A到C的最短：1（直接）
    3. 从A到D的最短 = min(从A到B最短+从B到D, 从A到C最短+从C到D)
                   = min(3+1, 1+4)
                   = min(4, 5)
                   = 4
    
    大问题的最优解包含小问题的最优解！
    */
}
```
