对数的应用场景是 把 复杂的乘除转为加减
 
a*a*a = a^3 , 指数3 等于 log_a (a3) = 3 // 消除log的底数 就得到 指数 （log必须有底数）
 

加法 乘法 乘方 指数 本质是 数在增大
减法 除法 开方 对数 本质是 数在减小

a+b-b = a; >>> 5+3-3=5; // 先加再减.   
a-b+b = a; >>> 5-3+3=5; // 先减再加

a*b/b = a; >>> 5*3/3=5; // 先乘再除
a/b*b = a; >>> 5/3*3=5; // 先除再乘

√(a²) = a; >>> √(5²) = 5; // 先乘方再开方 （2次方根） 指数和方根 相等 就可以抵消
(√a)² = a; >>> (√5)² = 5; // 先开方再乘方 （2次方根） 指数和方根 相等 就可以抵消

logₐ(aᵇ) = b; >>> log₂(2³) = 3; // 先指数再对数  底数和对数相等 就可以抵消  （读作 log以2为底 2的3次方）
a^(logₐ(c)) = c; >>> 2^(log₂ (3)) = 3;  // 先对数再指数 当对数作为指数时 对数的底数和真数相同就可以抵消
