## MD5 是什么？

**MD5** 是一种**哈希算法**，全称是 Message-Digest Algorithm 5（消息摘要算法第5版）。它可以将任意长度的数据，计算成一个固定长度（128位/32个十六进制字符）的"指纹"。

## MD5 的核心特点

### 1. **固定长度输出**
```javascript
// 无论输入多长，输出都是32位十六进制
md5('a')          // 0cc175b9c0f1b6a831c399e269772661 (32位)
md5('hello')      // 5d41402abc4b2a76b9719d911017c592 (32位)
md5('这是一本100万字的小说...') // 也是32位
```

### 2. **不可逆性**
```javascript
// 从结果无法反推原文
const hash = '5d41402abc4b2a76b9719d911017c592';
// 无法知道原文是 'hello'

// 就像把鸡蛋做成煎蛋，无法从煎蛋变回鸡蛋
```

### 3. **雪崩效应**
```javascript
// 输入微小变化，输出巨大变化
md5('hello')      // 5d41402abc4b2a76b9719d911017c592
md5('hello!')     // e0c90358968dd68f5b2b3f6b5f0d2c3e (完全不同)

// 就像：改一个字，整个指纹都变了
```

## MD5 的工作原理（简化版）

```javascript
// 极简模拟 MD5 处理过程
function simpleMD5(input) {
  // 1. 填充：把输入补足到固定长度
  let padded = padTo512Bits(input);
  
  // 2. 分组：每512位一组
  let groups = splitToGroups(padded);
  
  // 3. 处理每组数据（复杂的位运算）
  let result = initialValue;
  groups.forEach(group => {
    result = complexOperation(result, group);
  });
  
  // 4. 输出32位十六进制
  return toHexString(result);
}
```

## 实际应用场景

### 1. **文件完整性校验**
```javascript
// 下载文件时检查是否完整
const crypto = require('crypto');
const fs = require('fs');

// 计算下载文件的 MD5
function checkFileIntegrity(filePath, expectedMD5) {
  const fileContent = fs.readFileSync(filePath);
  const actualMD5 = crypto
    .createHash('md5')
    .update(fileContent)
    .digest('hex');
  
  if (actualMD5 === expectedMD5) {
    console.log('✅ 文件完整');
  } else {
    console.log('❌ 文件已损坏');
  }
}

// 使用
checkFileIntegrity('setup.exe', '8a2b3c4d5e6f7g8h...');
```

### 2. **密码存储（不推荐，但常见）**
```javascript
// ❌ 错误做法：直接存密码
db.save({ username: 'admin', password: '123456' });

// ⚠️ 旧系统做法：存 MD5
const md5 = crypto.createHash('md5').update('123456').digest('hex');
db.save({ username: 'admin', password: md5 }); // 'e10adc3949ba59abbe56e057f20f883e'

// ✅ 现代做法：加盐哈希
const bcrypt = require('bcrypt');
const hash = await bcrypt.hash('123456', 10);
```

### 3. **webpack 生成文件指纹**
```javascript
// webpack 内部使用 MD5 生成 hash
const hash = crypto
  .createHash('md5')
  .update(fileContent)
  .update(config)  // 加入配置信息
  .digest('hex')
  .slice(0, 8);    // 取前8位

// 输出: main.8a2b3c4d.js
```

### 4. **重复文件检测**
```javascript
// 找出重复的图片
const files = [
  'photo1.jpg', 'photo2.jpg', 'photo3.jpg'
];

const md5Map = new Map();

files.forEach(file => {
  const content = fs.readFileSync(file);
  const hash = crypto.createHash('md5').update(content).digest('hex');
  
  if (md5Map.has(hash)) {
    console.log(`⚠️ 重复文件: ${file} 和 ${md5Map.get(hash)}`);
  } else {
    md5Map.set(hash, file);
  }
});
```

### 5. **缓存键生成**
```javascript
// 根据请求参数生成缓存键
function generateCacheKey(params) {
  const str = JSON.stringify(params);
  return crypto.createHash('md5').update(str).digest('hex');
}

// 使用
const cacheKey = generateCacheKey({ userId: 123, page: 1 });
// cacheKey = '8a2b3c4d5e6f7g8h...'
```

## MD5 的安全性

### 1. **为什么 MD5 不安全了**
```javascript
// 2004年，中国密码学家王小云发现碰撞攻击
// 可以找到两个不同内容，但 MD5 相同

// 示例（理论上可行）：
const file1 = '正常合同.pdf';
const file2 = '恶意合同.pdf';  // 内容不同，但 MD5 相同
```

### 2. **碰撞攻击演示**
```javascript
// 两个不同程序，但 MD5 相同
program1.exe (正常软件)
program2.exe (病毒)  // MD5 和 program1 一样

// 用户校验 MD5 发现匹配，以为是正常软件
```

### 3. **彩虹表攻击**
```javascript
// 预计算的密码-MD5对照表
const rainbowTable = {
  '5f4dcc3b5aa765d61d8327deb882cf99': 'password',
  'e10adc3949ba59abbe56e057f20f883e': '123456',
  // ... 几十亿条记录
};

// 拿到 MD5 直接查表得到密码
```

## 现代替代方案

| 算法 | 输出长度 | 安全性 | 用途 |
|------|---------|--------|------|
| MD5 | 128位 | ❌ 不安全 | 文件校验、webpack hash |
| SHA-1 | 160位 | ⚠️ 已部分攻破 | Git 版本控制 |
| SHA-256 | 256位 | ✅ 安全 | 密码哈希、证书 |
| SHA-3 | 可变 | ✅ 最安全 | 政府、军事 |
| bcrypt | 可变 | ✅ 安全 | 密码存储 |

## 在 Node.js 中使用 MD5

```javascript
const crypto = require('crypto');

// 1. 计算字符串 MD5
function md5(str) {
  return crypto
    .createHash('md5')
    .update(str)
    .digest('hex');
}

console.log(md5('hello')); 
// 5d41402abc4b2a76b9719d911017c592

// 2. 计算文件 MD5
function md5File(filePath) {
  const content = fs.readFileSync(filePath);
  return crypto
    .createHash('md5')
    .update(content)
    .digest('hex');
}

// 3. 流式处理大文件
function md5LargeFile(filePath) {
  return new Promise((resolve, reject) => {
    const hash = crypto.createHash('md5');
    const stream = fs.createReadStream(filePath);
    
    stream.on('data', data => hash.update(data));
    stream.on('end', () => resolve(hash.digest('hex')));
    stream.on('error', reject);
  });
}

// 4. 加盐（增加安全性）
function md5WithSalt(str, salt) {
  return crypto
    .createHash('md5')
    .update(str + salt)
    .digest('hex');
}
```

## 在浏览器中使用 MD5

```html
<!-- 使用 spark-md5 库 -->
<script src="https://cdn.jsdelivr.net/npm/spark-md5@3.0.2/spark-md5.min.js"></script>

<script>
// 计算字符串 MD5
const hash = SparkMD5.hash('hello');
console.log(hash); // 5d41402abc4b2a76b9719d911017c592

// 计算文件 MD5
function calculateFileMD5(file) {
  return new Promise((resolve, reject) => {
    const chunkSize = 2 * 1024 * 1024; // 2MB
    const spark = new SparkMD5.ArrayBuffer();
    const reader = new FileReader();
    
    let currentChunk = 0;
    const chunks = Math.ceil(file.size / chunkSize);
    
    reader.onload = (e) => {
      spark.append(e.target.result);
      currentChunk++;
      
      if (currentChunk < chunks) {
        loadNext();
      } else {
        resolve(spark.end());
      }
    };
    
    reader.onerror = reject;
    
    function loadNext() {
      const start = currentChunk * chunkSize;
      const end = Math.min(start + chunkSize, file.size);
      reader.readAsArrayBuffer(file.slice(start, end));
    }
    
    loadNext();
  });
}

// 使用
document.querySelector('input[type="file"]').onchange = async (e) => {
  const file = e.target.files[0];
  const hash = await calculateFileMD5(file);
  console.log('文件 MD5:', hash);
};
</script>
```

## MD5 在 webpack 中的具体应用

```javascript
// webpack 内部简化代码
class WebpackMD5 {
  generateHash(modules) {
    // 1. 收集所有模块内容
    const allContent = modules
      .map(m => m.content)
      .join('');
    
    // 2. 加上配置信息
    const withConfig = allContent + JSON.stringify(this.config);
    
    // 3. 计算 MD5
    const hash = crypto
      .createHash('md5')
      .update(withConfig)
      .digest('hex');
    
    // 4. 取前8位作为文件名
    return hash.slice(0, 8);
  }
}

// 使用
const hash = generateHash(modules);
// 输出: main.8a2b3c4d.js
```

## 总结

**MD5 就是一个"文件指纹生成器"**：

- ✅ **优点**：速度快、实现简单、输出固定长度
- ❌ **缺点**：不安全（可碰撞）、不适合密码存储
- 📌 **现在用途**：文件完整性校验、webpack hash、重复文件检测
- 🚫 **不要用于**：密码存储、数字签名、安全敏感场景

在 webpack 中，MD5 仍然是生成 hash 的主流选择，因为：
1. 速度快，不影响构建性能
2. 只需要唯一性，不需要密码级安全
3. 输出长度适中，32位十六进制刚好
