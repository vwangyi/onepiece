# 变量  
## 概念名词 
- 变量：用来操作内存的
- 常量就是不能修改的变量。 用const声明，引用类型的变量只要引用地址不变就不算修改。
- 暂时性死区 
- 变量提升：变量提升是声明会提升 赋值不会提升。 
- 块级作用域是 除了对象的花括号 其他{} 都算。用let const 声明都有块级作用域。
- 大驼峰 
- 小驼峰 
- 标识符identifier 是 用来命名变量的符号， 由字母 数字 下划线(_)组成， 不能数字开头 不能是关键字 保留字，区分大小写。
- 空白符：增加代码可读性，比如 空格 回车 
- 关键字keyword：已经被官方使用的符号
- 保留字：官方目前没使用，未来可能会使用的符号。
- js变量没有类型，数据才有类型，ts是让js变量有类型。
- 数据类型有哪些：数据类型是7个基础类型1个引用类型。分别是 number string boolean null undefined symbol bigint 和 object。
- 工厂函数是 创建并返回实例对象的函数，好处是封装了复杂性 私有性。对于 用类创建对象和工厂函数创建对象的区别 vue也是用 createApp工厂函数来创建vue实例对象
- JS的6大假值是什么： false '' null undefined 0 NaN 
- 柯里化是把多参函数优化为嵌套的单参函数 是什么 应用 柯里化 固定化参数 好处 

- 帧率是 
我的显示器是60Hz（HZ表示硬件显示器）
这个游戏能跑到60fps （fps表示软件） 
60帧 既表示60Hz 也表示60fps
60HZ 或 60fps 
60赫兹(HZ)  表示 每秒画60次， 那么画一帧的耗时就是 1000/60 = 16.67毫秒 ms （一帧就是一张图片）也就是 每16.67ms 渲染一次


requestAnimationFrame 可以让回调在 下一次渲染前调用 使用场景是
requestIdleCallback 可以让 回调在 主线程空闲之前调用 

- 循环是 重复执行同一段代码
- 递归是函数自己调用自己的行为
- 遍历是 按照某种顺序 操作数据结构中的每一个元素 操作往往包含 curd
- 迭代是 按照某种顺序 访问数据结构中的每一个元素 迭代属于遍历的一种 

- 可迭代对象：对象的原型链上有 迭代器对象，那就是可迭代的对象，比如 Array Map Set 的原型链上 都有迭代器对象， 而 Object的原理链上没有除非手动添加
- 迭代器对象是什么 ：迭代器对象是有 next方法 next方法返回value和done的对象  done表示是否可以继续调用next()  一般不会手动创建迭代器对象 而是用生成器函数默认返回迭代器对象
- 生成器函数是用function* 声明的函数，内部可以使用 yeild 关键字

- 有生成器函数或迭代器对象的数据结构 可以 用 forof循环

- async函数和aawit表达式 底层原理就是 生成器函数 + yield表达式 和 co模块


- 单线程：js是单线程语言 因为js只能被浏览器的主线程执行，而主线程只有一个但浏览器是一个多进程多线程的应用程序。

- 同步任务：同步任务就是 主线程立即执行 不用排队
- 异步任务：异步任务 是主线程不能立即执行的任务，交给其他线程当异步任务满足触发条件就会进入任务队列进行排队等待主线程执行。比如 IO 网络等 

> js变量没有类型，只有数据才有类型。ts则是让变量拥有类型。

## var let const有什么区别
1. var和let的区别是 let 有块级作用域，所以没有全局污染 没有变量提升所以有暂时性死区
2. let const 唯一区别是 const必须赋值且不能修改。

```js
let a; // 变量声明 
let b = 1, c = 2; // 变量声明并赋值 
```


变量提升 面试题
```js
var a = 1; // var变量a
function a() {  // function变量a 
  console.log(a);
  let a = 2;
}
console.log(a); // 输出 1 
a(); // 报错 a不是一个函数

// 函数声明提升到顶部，a 最初是函数
// var a 覆盖了 function a 
// 所以 a为1 a不是一个函数

```
