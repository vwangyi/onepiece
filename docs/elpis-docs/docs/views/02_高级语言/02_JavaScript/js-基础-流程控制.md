# 流程控制语句 
- 顺序 分支 循环 

在JavaScript中，‌顺序结构‌是最基本的流程控制方式，指的是程序中的代码按照‌从上到下、从左到右‌的顺序依次执行，每一行语句在前一行执行完毕后自动执行，没有跳转或分支。

这种结构是JavaScript引擎的默认执行方式，不需要任何特殊的语法关键字。程序中的大多数代码都属于顺序结构，例如变量声明、赋值、函数调用和输出语句等，都会按其在代码中出现的先后顺序逐一执行。

特点
‌无条件执行‌：代码按书写顺序逐行执行，不依赖任何条件判断。
‌线性执行‌：上一行代码的执行结果（如变量赋值）会直接影响下一行代码的行为。
‌基础性‌：是分支结构和循环结构的基础，所有复杂控制逻辑都建立在顺序执行之上。

## 是什么
- 流程控制语句-顺序结构：代码‌从上到下、从左到右‌的顺序，一行一行依次执行.


```js
var str = "hello";
var str1 = " world";
var str2 = str + str1;
console.log(str2); // 输出: "hello world"

// 执行顺序：
// 执行 var str = "hello";
// 执行 var str1 = " world";
// 执行 var str2 = str + str1;
// 执行 console.log(str2);
```

```js
if () {}
if () {} else {}

三元表达式 
 

switch( 表达式 ){ 
 case value1:
    // 表达式 等于 value1 时要执行的代码
 break;
 case value2:
    // 表达式 等于 value2 时要执行的代码
 break;
 default:
    // 表达式 不等于任何一个 value 时要执行的代码
}
switch ：开关 转换 ， case ：小例子 选项
关键字 switch 后面括号内可以是表达式或值， 通常是一个变量
关键字 case , 后跟一个选项的表达式或值，后面跟一个冒号
switch 表达式的值会与结构中的 case 的值做比较
如果存在匹配全等(===) ，则与该 case 关联的代码块会被执行，并在遇到 break 时停止，整个 switch 语句代码 执行结束
如果所有的 case 的值都和表达式的值不匹配，则执行 default 里的代码
```

switch 语句和 if else if 语句的区别
① 一般情况下，它们两个语句可以相互替换
② switch…case 语句通常处理 case为比较确定值的情况， 而 if…else…语句更加灵活，常用于范围判断(大于、等于某个范围)
③ switch 语句进行条件判断后直接执行到程序的条件语句，效率更高。而if…else 语句有几种条件，就得判断多少次。
④ 当分支比较少时，if… else语句的执行效率比 switch语句高。
⑤ 当分支比较多时，switch语句的执行效率比较高，而且结构更清晰。

# 循环

循环是重复执行一段代码

for 循环
多重for循环 
while 循环
do…while 循环

```js
for(初始化变量; 条件表达式; 操作表达式 ){
 //循环体
}

do {
 // 循环体代码 - 条件表达式为 true 时重复执行循环体代码
} while(条件表达式);

 
continue; 关键字用于立即跳出本次循环，继续下一次循环（本次循环体中 continue 之后的代码就会少执行一次）。
 
break; 关键字用于立即跳出整个循环（循环结束）。
```

迭代属于循环的一种

循环和递归 可以互转

```js
const arr = [1,2,3]
function sum(arr, i = 0) {
   if(i>= arr.length) return 0;
   return arr[i] + sum(arr, i + 1)
}
console.log(sum(arr)); // 6

所有的循环都可以转 for循环

for(初始代码;条件代码;循环代码) {
   // 循环体
}

function m() {
   初始代码;
   function _m() {
      if(!条件代码) {
         return;
      }
      循环体;
      循环代码;
      _m();
   }
   _m()
}
```


循环：侧重于对代码块的重复执行

遍历”（traversal）则适用于对非线性结构的数据进行特定顺序的访问。例如，在树、二叉树或图等数据结构中，我们可以根据需要选择不同的遍历方式，如先根遍历、中根遍历或后根遍历等。

“迭代”（iterate）特指对线性结构中的数据按照先后顺序进行访问。线性结构如数组，其元素只能从头到尾或从尾到头依次访问，这种访问方式是天然决定的。

“递归”（recursion）是指函数对自身的不断调用。
