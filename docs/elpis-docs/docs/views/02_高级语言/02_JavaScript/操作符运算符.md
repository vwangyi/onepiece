> 逻辑运算符 返回满足条件的值
>

```javascript
&& || !  与 或 非

// 什么是逻辑中断？
// 逻辑运算符是双元运算符 即只有两个操作数
// 由于第一个操作数的真假 就可以决定整个表达式的真假 
// 而不需要对第二个操作数进行真假判定 这就是逻辑中断 

// 为什么返回123？ 
// 因为&&是且 表示 两者全都满足 
// 首先对[]进行真假判断 发现为真 由于逻辑中断 直接返回后者（不会对后者进行真假判定）
const a = [] && 123 // 123 
// 为什么返回''？
// 因为||是或 表示 两者满足一个即可
// 首先对0进行真假判断 发现为假 由于逻辑中断 直接返回后者（不会对后者进行真假判定）
const b = 0 || '' // ''
 
const c = !0  // true   对操作数进行真假判定 然后取反 返回的是boolean值
const d = ![] // false   



[]&&123 // 当前者为真 返回后者 前者为假 返回前者
0||''   // 当前者为假 返回后者 前者为真 返回前者


&&的应用场景：1 逻辑判断真假 2 逻辑中断

if([]&&123) {
  console.log('真');
} else {
  console.log('假');
}
const a = callback && callback(); // 如果callback存在就执行callback

if(1||[]){
  console.log('真');
} else {
  console.log('假');
}
const b = callback || '123'; // 如果callback不存在 默认值为'123'
```

>

