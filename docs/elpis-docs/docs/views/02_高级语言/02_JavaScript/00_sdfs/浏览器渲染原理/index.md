## 浏览器执行js代码

```js
// 浏览器有渲染引擎和js引擎(V8)
// 渲染引擎(浏览器内核)：用于渲染html和css
// js引擎(js解析器)：比如chrome的v8 用于读取网页js代码

// JavaScript是运行在客户端的脚本语言
// 脚本语言不需要编译，运行过程中由js解释器(js引擎)从上至下逐行来进行解释并执行
// 现在也可以基于Node.js技术进行服务器端编程

// js引擎把 高级语言 解析为 二进制语言 供浏览器执行

// 高级语言  => js引擎 => 二进制 => 执行

// 编译:java是编译型语言  把java文件 编译为 class文件(中间代码文件) 再编译为机器语言
// 解释:javascript是解释型语言  html文件 通过js解析器  一行一行 从上至下 解释为机器语言
```
