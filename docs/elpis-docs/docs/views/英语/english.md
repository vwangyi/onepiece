## 6 名词的可数不可数
可数名词: 个体名词 集体名词

不可数名词: 物质名词 抽象名词

可数名词单数: 前面 通常 要 + a 或者 an    a用在 辅音音素开头前的单词 an用在 元音音素开头前的单词 (辅音音素就是辅音音标 元音音素就是元音音标)  
在现阶段 可以简单的理解为: a用在辅音字母前 an用在元音字母a e i o u 前

做题

___pen      a pen  
___dag      a dag  
___apple   an apple

___big apple  a big apple  
___egg        an egg  
___orange   an orange  
___banana   a banana  
___English  an English

特例  
___useful book  a useful book    (u开头的单词 读音是 元音音素/j/   但是不是元音字母 所以用a)

可数名词复数: 后面通常 要 + s

复数规则变化  
  1 直接加s 比如: cat cats apples (除了其他规则 就是直接+s)  
  2 以s x sh ch 结尾 加es 比如: bus >>> buses  box >>> boxes peach >>> peaches brush >>> brushes  
  3 最后以辅音字母 + y 结尾   变y为i 再加es 比如: baby >>> babies  family >>> families   story >>> stories  
  4 最后以元音字母 + y 结尾 直接 +s   boy >>> boys day >>> days  key >>> keys  
  5 以o结尾 有生命 加es 没有生命 直接 +s 比如: potato >>> potatoes  photo >>> photos  
  6 以f/fe结尾  去f/fe   + ves 比如 knife >>> knives  leaf >>> leaves

名词的不规则变化  
  1单复数同行(单复数长一样)   
    比如: sheep >>> sheep  fish >>> fish  deer >>> deer  Chinese >>> Chinese  Japanese >>> Japanese Swiss >>> Swiss  
  2变元音
